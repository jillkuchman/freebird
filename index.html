Adopt an Epicoder

This is a Drupal website for the mock charity Adopt an Epicoder, where users can donate to fund junior developers. The site is based off of the Commerce Kickstart package and uses a custom theme created from the Omega theme. The website includes a display of products by product type, and a taxonomy-based menu to organize and display content.

  • Client: Epicodus Group Project
  • Date: June 2015
  • Technologies: Drupal, PHP, CSS

Order A Pizza

This project mimics a website that allows a user to order a pizza. Users can select different options for toppings, crust, and size, and the app calculates the price based on selection.

  • Client: Epicodus/Personal
  • Date: April 2015
  • Technologies: JavaScript, jQuery, CSS, HTML, Bootstrap

Tamacrotchety

This project emulates a Tamagotchi game. The user creates a crotchety Tamagotchi-like creature with health levels and can interact with it in various ways to keep it alive. This project demonstrates writing methods for objects in JavaScript and calling those methods using jQuery.

  • Client: Epicodus/Personal
  • Date: April 2015
  • Technologies: JavaScript, jQuery, HTML, CSS, Bootstrap

Epifoodus

This project removes the decision making from choosing a place to eat by randomly suggesting two restaurants from a database. Users can create a profile and save preferences. I handled user profile creation with password verification.

  • Client: Epicodus/Personal
  • Date: March 2015
  • Technologies: PHP, PSQL, HTML, CSS, Bootstrap
